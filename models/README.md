# 3D 模型文件夹

## 说明

此文件夹用于存放 `.obj` 格式的 3D 模型文件。

## 使用方法

1. 将您的 `.obj` 文件放置在此文件夹中
2. 在网页应用中，通过「数据源接入」上传对应的 `.obj` 文件
3. 系统会自动进行粒子化重构并显示

## 支持格式

- ✅ `.obj` 文件（无需贴图）
- ⚠️ 建议模型顶点数在 10,000 - 100,000 之间以获得最佳性能

## 示例模型来源

您可以从以下网站获取文物 3D 模型：
- [Sketchfab](https://sketchfab.com/) - 搜索 "cultural heritage" 或 "artifact"
- [Smithsonian 3D](https://3d.si.edu/) - 史密森尼博物馆 3D 数字馆藏
- [CyArk](https://www.cyark.org/) - 文化遗产数字化项目

## 注意事项

- 文件大小建议不超过 50MB
- 复杂模型可能需要较长的加载和处理时间
- 系统会自动对模型进行居中和缩放处理
